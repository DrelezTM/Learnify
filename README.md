## How to run Front-End ðŸ“‘
* Open Front-End Folder
  ```sh
  cd FE-Learnify
  ```
* Install Dependencies
  ```sh
  npm install
  ```
* Starting
  ```sh
  npm run dev
  ```

## How to run Back-End ðŸ“‘
* Open Back-End Folder
  ```sh
  cd BE-Learnify
  ```
* Install Dependencies
  ```sh
  composer install
  ```
* Migrating Database
  ```sh
  php artisan migrate
  ```
  ```sh
  php artisan db:seed
  ```
* Linked Storage
  ```sh
  php artisan storage:link
  ```
* Starting
  ```sh
  php artisan serve
  ```
